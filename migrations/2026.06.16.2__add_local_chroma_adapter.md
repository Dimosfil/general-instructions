# Add Local Chroma Adapter

Migration ID: `2026.06.16.2__add_local_chroma_adapter`

## Summary

Add an optional local Chroma adapter that builds and queries a generated vector
index from exported project-memory chunks.

## Accepted Guidance

- Use Chroma as an optional local retrieval adapter, not as the whole RAG
  system.
- Build Chroma from generated semantic chunks exported by
  `build_project_memory_index.py export-chunks`.
- Keep the generated Chroma index ignored and rebuildable.
- Use `uv run --with chromadb` when the project does not commit Python
  dependencies for the adapter.
- Keep keyword retrieval available for exact paths, commands, symbols, versions,
  and errors.
- Keep collection names and embedding metadata in `rag-system.json`; keep
  credentials and private runtime endpoints out of committed files.

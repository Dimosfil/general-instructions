# Add Skill Modules

Migration ID: `2026.05.16.1__add_skill_modules`

## Purpose

Add the reusable skill-module concept to projects that copied this instruction
kit.

## Shared Source Files

- `patterns/SKILL_MODULES.md`
- `templates/SKILL.template.md`

## Project Files

- Optional: `skills/TODO-skill-name/SKILL.md`
- Existing project rules may mention `patterns/SKILL_MODULES.md` when deciding
  whether a capability should be a rule, pattern, template, or skill.

## Apply

- Copy `templates/SKILL.template.md` only when the project needs a concrete
  skill.
- Do not create empty skill folders by default.
- Add a short reference to skill modules in project working agreements only when
  the project expects reusable skills.

## Verify

- Confirm no secrets or project-private data are added to shared skills.
- Confirm project-local skills are owned by the project, not linked to this
  shared repository.

# Modularize Agents Runtime Entrypoint

Migration ID: `2026.06.21.13__modularize_agents_runtime_entrypoint`

## Summary

Replace the monolithic copied `AGENTS.md` runtime guidance with a compact
entrypoint and topic-routed modules under `patterns/AGENTS_RUNTIME/`.

The accepted behavior stays the same: agents must still follow the same startup,
scope, safety, command, language, verification, and git rules. The change is the
retrieval shape: agents read the entrypoint first, then load only the modules
that match the current task.

## Scope

- `AGENTS.md`
- `README.md`
- `INDEX.md`
- `CHANGELOG.md`
- `VERSION.md`
- `patterns/AGENTS_RUNTIME/`
- `templates/AGENTS.template.md`
- `tools/project-memory/architecture-migrations.md`
- `tools/project-memory/instruction-kit.json`
- `templates/instruction-kit.template.json`

## Accepted Guidance

- Keep root `AGENTS.md` and project-local `AGENTS.md` files as lightweight
  runtime entrypoints.
- Put detailed reusable runtime rules in focused modules under
  `patterns/AGENTS_RUNTIME/`.
- Include a module routing table in each entrypoint so agents know which files
  to read for startup, task-manager work, deploy, rebuild, language, UI,
  update, verification, and git tasks.
- Preserve compatibility with the former monolithic guidance. Splitting files
  must not weaken safety, scope, command, verification, or release rules.
- Include the runtime modules in instruction-kit copied files so consuming
  projects receive both the entrypoint and its linked rule modules.

## Verification

- Confirm root `AGENTS.md` is compact and links to every runtime module.
- Confirm `templates/AGENTS.template.md` is compact and routes to copied
  modules.
- Confirm `templates/instruction-kit.template.json` and
  `tools/project-memory/instruction-kit.json` list every
  `patterns/AGENTS_RUNTIME/*.md` module in `copied_files`.
- Confirm no accepted rule is intentionally removed; behavior changes should be
  limited to module routing and context loading.
- Run documentation checks such as `git diff --check`.

## Rollback

Restore the previous monolithic `AGENTS.md` and `templates/AGENTS.template.md`,
remove `patterns/AGENTS_RUNTIME/` from copied files, and reset the instruction
kit version metadata to the previous accepted version.

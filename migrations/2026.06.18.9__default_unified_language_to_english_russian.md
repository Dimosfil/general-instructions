# Default Unified Language To English Russian

Migration ID: `2026.06.18.9__default_unified_language_to_english_russian`

## Purpose

Make the unified `gi language` / `gi язык` flow default to the first two
language choices when a project has no current ordered selection for a language
surface.

## Merge Rules

- When `gi language`, `gi язык`, `ги язык`, or an equivalent unified
  project-language command is sent without explicit languages, keep the
  existing three-step language flow.
- If a language surface has no current ordered selection, show and use `1 2` as
  the default ordered selection: `English`, then `Russian`.
- Preserve explicit existing selections. Do not overwrite a real current
  project, commit-message, or task language order merely because this default
  exists.
- Keep numeric replies such as `1 2` mapped to the most recent checklist.
- Update manual project-language selectors so empty input with no existing
  preference saves the same `English`, then `Russian` order.

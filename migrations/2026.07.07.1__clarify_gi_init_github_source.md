# Clarify GI Init GitHub Source

Migration ID: `2026.07.07.1__clarify_gi_init_github_source`

## Summary

Make `gi init` with the canonical GitHub source explicit. The full URL,
shorter `Dimosfil/general-instructions.git` form, and Markdown links to either
form mean shared-instruction bootstrap, not ordinary git repository
initialization or current-project remote replacement.

## Instructions

- Treat `gi init https://github.com/Dimosfil/general-instructions.git`,
  `gi init Dimosfil/general-instructions.git`, and Markdown links to that repo
  as shared-instruction bootstrap/startup requests.
- Resolve the source as the shared instruction library and deploy or restore
  the local GI instruction kit according to the bootstrap contract.
- Do not run ordinary `git init`, replace the current project's remote, create
  a submodule, add a dependency, or create a thin pointer-only `AGENTS.md`
  unless the user explicitly asks for that separate git action.

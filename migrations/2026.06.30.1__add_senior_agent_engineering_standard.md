# Add Senior Agent Engineering Standard

## Summary

Add a compact senior-engineering execution standard that connects existing GI
rules for code-writing agents into one discoverable pattern.

## Accepted Changes

- Add `patterns/SENIOR_AGENT_ENGINEERING_STANDARD.md`.
- Link the new pattern from `README.md` and `INDEX.md`.
- Reference the standard from the runtime content/authoring module so agents can
  route broad code-quality and reusable-rule work to the compact checklist.
- Bump the accepted instruction-kit version to `2026.06.30.1`.

## Agent Guidance

When writing, reviewing, or refactoring code, agents should act as maintainers:
load relevant local context, preserve intended behavior, keep architecture and
configuration boundaries clear, work in coherent verified batches, update
durable project memory for meaningful behavior or architecture changes, and
escalate high-risk actions through documented approvals.

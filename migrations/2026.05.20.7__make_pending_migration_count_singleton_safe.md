Migration ID: `2026.05.20.7__make_pending_migration_count_singleton_safe`

## Purpose

Prevent the instruction-kit update helper from failing when exactly one pending
migration exists.

## Apply

- Wrap the pending migration pipeline in an array expression:

  ```powershell
  $pending = @(Get-ChildItem ... | Where-Object { ... })
  ```

- Preserve the existing pending migration filters, sort order, and metadata
  recording behavior.
- Apply the same singleton-safe pattern in any copied instruction-kit helper
  script that counts pipeline results under strict PowerShell mode.

  migrations when practical.
- Confirm a single pending migration reports count `1` instead of failing with
  a missing `.Count` property.

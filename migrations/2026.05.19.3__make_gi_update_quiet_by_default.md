# Make GI Update Quiet By Default

## Purpose

Reduce token use and visible chatter during `gi обновить` / `gi update`.
Instruction-kit updates should mostly apply directly, then produce a compact
status report.

## Source Files

- `COMMANDS.md`
- `AGENTS.md`
- `patterns/INSTRUCTION_KIT_MIGRATIONS.md`
- `templates/AGENTS.template.md`
- `templates/AGENT_WORKING_AGREEMENTS.template.md`
- `templates/check-instruction-kit-updates.template.ps1`
- `templates/instruction-kit.template.json`

## Project Files To Update

- `AGENTS.md`
- `tools/AGENT_WORKING_AGREEMENTS.md`

## Merge Rules

- Treat `gi обновить` as quiet-by-default work.
- Do not narrate step-by-step reasoning, repeated progress updates, command
  transcripts, or broad file reads during a normal successful update.
- Use compact internal checks and surface only blockers, correctness-affecting
  warnings, or the final summary.
- Final summary should include only version before/after, pending/applied
  migration count, applied migration IDs, changed file count or names, checks
  run, commit/push result if applicable, and any blocker.
- Read only accepted artifacts required for the update: `VERSION.md`,
  `CHANGELOG.md`, targeted migration files, and required templates/patterns.
  Do not read `updates/`, broad repository files, full diffs, or large logs.
- Apply and commit only instruction-kit update files. Stop on unrelated local
  changes, old untracked artifacts, conflicts, secrets, or unclear scope.
- Keep a verbose/debug mode available for failure investigation or explicit
  user requests.

## Verification

- Run the project update-check script in default mode and confirm output is
  compact.
- Run with the verbose/debug option, if added, and confirm it provides migration
  filenames/details for troubleshooting.
- Reread changed instruction files.
- Run `git diff --check`.

## Rollback Or Conflict Notes

If a project needs more detailed update output for a one-off incident, use the
verbose/debug mode for that run only. Do not make verbose progress the default.

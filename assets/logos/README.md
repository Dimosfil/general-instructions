# General Instructions brand assets

`general-instructions-logo.png` is the primary symbol-only logo concept for the
General Instructions Library. The open pages represent reusable instruction
modules; the right-pointing arrow represents routing guidance into agent action.

`general-instructions-banner.png` is the 1600x320 dark-background banner for
the repository README and other wide GitHub presentation surfaces.

The source concept uses deep navy and cyan-blue on white. Keep the silhouette
uncluttered and do not add typography inside the mark.
